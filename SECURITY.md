# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Fullsite, please report it responsibly.

**Email:** seguridad@fullsite.mx
**Response time:** 48 business hours
**Language:** Spanish or English

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will not take legal action against security researchers acting in good faith.

## Scope

- app.fullsite.mx (dashboard and POS)
- API endpoints under app.fullsite.mx
- GitHub Actions workflows

## Out of Scope

- Third-party services (Supabase, Vercel, Cloudflare, Anthropic)
- Social engineering attacks
- Denial of service attacks
