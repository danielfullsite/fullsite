# Guía Mesero — Fullsite POS

## Login
1. Ingresa tu PIN de 4 dígitos
2. Toca **Entrar**

## Tomar orden
1. Selecciona la **Mesa** (número arriba a la izquierda)
2. Pon el **número de personas**
3. Toca una **categoría** (ej. Coffee Hot/Ice, Chilaquiles)
4. Toca el **producto** que quiere el cliente
5. Si tiene modificadores (sin cebolla, extra queso), márcalos y toca **Confirmar**
6. Repite para cada producto

## Mandar a cocina
1. Cuando tengas toda la orden, toca el botón verde **Cocina**
2. La comanda se imprime automáticamente en cocina y/o barra
3. El botón cambia a "Enviado" ✓

## Tiempos (si el cliente pide primero y segundo)
1. Agrega los productos del primer tiempo
2. Toca **Tiempo** para insertar un separador
3. Agrega los del segundo tiempo
4. Al tocar **Cocina**, se imprimen separados

## Cobrar
1. Toca el botón azul **Cobrar**
2. Confirma el número de personas
3. Selecciona la forma de pago:
   - **Efectivo**: ingresa la cantidad recibida, el sistema calcula el cambio
   - **Tarjeta**: selecciona crédito o débito
   - **Mixto**: agrega varios pagos hasta cubrir el total
4. El ticket se imprime y el cajón se abre (si es efectivo)

## Pre-cuenta
- Toca **Cuenta** (botón naranja) para imprimir la cuenta sin cobrar

## Importante
- **NO cierres la ventana de Chrome** — el POS vive ahí
- **NO cierres la ventana negra (CMD)** — es el puente de impresión
- Si algo no funciona, avisa al gerente
- El POS funciona sin internet — las órdenes se guardan y sincronizan después
